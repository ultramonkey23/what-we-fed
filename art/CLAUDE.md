# CLAUDE.md - Art and VFX

VFX should sell impact without hiding timing truth.

- **Player Form**: All visuals MUST align with the implemented [Player Vessel Codex](../docs/LIVING_CODEX_PLAYER_VESSEL.md).
- Enemy pressure should read hotter than support; support should stay distinct and legible.
- Do not obscure lanes, projectiles, hit confirms, parry windows, or player hurt state.
- For visual quality claims, use `docs/ai/VISUAL_TRUTH_LOOP.md` or state that the pass is static-only.
- Run `validate_project.bat` after shader/import/resource edits.
