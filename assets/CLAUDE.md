# CLAUDE.md - Assets

Assets must improve readability and identity, not just decoration.

- **Player Form**: All new assets MUST align with the [Player Vessel Codex](../docs/LIVING_CODEX_PLAYER_VESSEL.md).
- Preserve import sidecars and Godot resource paths.
- Combat-facing art must keep threats, lanes, player response, and support effects readable.
- Prefer concrete visual evidence before replacing or tuning assets.
- Run `validate_project.bat` after asset/import changes.
