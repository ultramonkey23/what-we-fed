# WHAT WE FED — FINAL GAME SCOPE CANON (FLAGSHIP VERSION)

## Purpose

This file defines the **true flagship scope** of the game.
It should be treated as the project’s highest-level spine.

---

# 1. Final Game Identity: Dark RPG Roguelite

- **Shorthand**: combat-clean, management-rich.
- **Soul**: Start weak, become feared.
- **World Origin**: Wondrous, strange, and sacred start; diverges through consequence.

---

# 2. Reward Ecology: Distinct Lanes

 Progression lanes must stay distinct to avoid stat sludge:

1.  **Loot / Gear**: Shapes the hunter. Creature-derived trophies, organs, carapace, grafts.
2.  **Artifacts / Relics**: Shape the run identity. Rare, transformative, synergy-rich.
3.  **DNA**: Foundational species economy earned from actual kills.
4.  **Bond / Eat**: The core identity split (relational vs predatory).
5.  **Collars**: Shape bonded support behavior (doctrine/harness tech).
6.  **Tendencies**: Behavior-shaped growth spine.

---

# 3. World Fate Divergence

Repeated runs bend the world across four fate channels. The world reacts to player philosophy of power.

- **Predatory / Brutal**: Hunger made law. Carcass landmarks, red-black menace.
- **Mythic / Hopeful**: Costly endurance. Bruised-gold relic geometry, dangerous reverence.
- **Sterile / Technocratic**: Cold authority. Wrong-clean grids, invasive optimization.
- **Haunted / Ritual**: Debt and afterimage. Offerings, wrong shadows, ritual residue.

---

# 4. Core Design Spine

## Pillar 1 — Combat Timing Trust
Timing truth is non-negotiable.

## Pillar 2 — Lane Pressure
Three lanes for reading pressure and prioritization.

## Pillar 3 — Bond vs Eat
Immediate predation vs long-term relational power.

## Pillar 4 — Reward Discipline
**Loot + Artifacts + Collars** are separate schemas. No generic trash.

## Pillar 5 — Bosses as Battlefield Law
Bosses function as decree moments that shift arena laws.

## Pillar 6 — Kaiju Ascension (The Hollow Egg)
A long-term consequence engine shaped by what the player "taught" the world.

---

# 5. Operational Loop Split

- **Pre-run**: Strategic planning.
- **In-level Combat**: Combat-clean, pressure-first, stripped HUD.
- **Between-level**: Management-rich, thoughtful build-shaping.
- **Return to Combat**: Restore urgency quickly.

---

# 6. Anti-Drift Rule

The game dies if it becomes:
- a generic survivor clone
- a generic pet collector
- a stat spreadsheet
- a content-bloated mess

**Preserve timing trust, creature meaning, and the wonder-to-consequence world arc.**
