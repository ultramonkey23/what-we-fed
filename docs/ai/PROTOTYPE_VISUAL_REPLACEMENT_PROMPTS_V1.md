# Prototype Visual Replacement Prompts v1

Use these prompts for next-wave art generation while preserving gameplay readability.

## Direction lock
- Dark mythic core + anime authority.
- Strong silhouettes at gameplay scale.
- Restrained accents: ember, mutation magenta, bond teal, rare hopeful highlights.
- Brief wonder/beauty/cute moments only on positive mastery/support beats.
- No clutter, no glowspam, no particle soup.

## Tier 1 - Gameplay critical

### Impact FX sheet (perfect / parry / miss / dodge / elite / boss)
`dark anime combat impact glyphs, sharp slash geometry, high authority timing punctuation, transparent background, readable at 128x128, one effect per frame, restrained accent colors, premium inked linework, no noisy particles`

### Projectile overlays (shot1..shot6)
`anime speedline projectile overlays, horizontal motion language, crisp silhouettes, transparent background, readable at 32x32, role-distinct forms (fang, mass, needle, veil, chorus, sovereign), minimal palette, no blur soup`

### Projectile bodies (species keys)
`dark fantasy creature projectile cores, species-specific silhouettes, readable at 32x32, transparent background, high contrast ink shading, functional gameplay icon clarity, no decorative clutter`

### Lane danger marker set
`combat lane telegraph markers, high readability, clear danger tiers, transparent background, sharp geometric language, dark premium style with restrained warning accents`

## Tier 2 - Identity visuals

### Reward icon strips (DNA / Quig)
`animated icon strip, 32x32 frames, dark premium UI iconography with subtle anime charm, clear readability at small scale, transparent background, DNA icon (mystic mutation), Quig icon (slight cute hopeful personality), no cartoon overload`

### Creature reward portrait cards
`creature portrait icons for combat reward panels, dark mythic manga look, threatening but elegant, occasional gentle spark of wonder for bonded/support creatures, transparent background, clear head/torso silhouette at small sizes`

## Tier 3 - Environment backgrounds

### Combat biome triplet
`mythic cursed battlefield backgrounds, anime cinematic composition, strong foreground-mid-background separation for lane readability, ruined silhouettes, restrained celestial wonder, dark premium palette, no visual noise over gameplay center`

## Shell / title support

### Title and route accent motifs
`minimal UI motif overlays for title and route screens, dark ink base with premium anime highlights, subtle hope cues, restrained cuteness hints in companion motifs, high legibility, no clutter`
