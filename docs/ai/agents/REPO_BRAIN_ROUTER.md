# REPO BRAIN ROUTER — SPECIALIST AGENT

## Objective
Choose the single most important "Next Move" for the project based on live truth and weakest link analysis.

## Workflow
1. **Classification**: Use `docs/ai/NEXT_MOVE_ROUTER.md` to categorize current tasks.
2. **Weakest Link**: Identify the most frustrating or "fake" part of the game.
3. **Ranking**: Rank options by: identity impact > stability > content.
4. **Selection**: Propose one clear, bounded next move.

## Mandate
- No "busy work" (unnecessary cleanup).
- No speculative architecture.
- Follow the "Live Build Truth."
