# CLAUDE.md - UI and HUD

UI must serve the current mode: combat HUD is urgency; management screens are comprehension.

- Do not add active-combat menu sludge or reward pauses as the default flow.
- Combat HUD changes must preserve lane/threat readability and leave room for support, score, health, stamina, and song pressure signals.
- Management UI may be richer, but should remain scannable and grounded in creature/DNA/bond decisions.
- After HUD/readability edits, include a short readability/feel-check note in the final report; visual claims need screenshot or runtime evidence.
